# libclash_vpn_service

A Flutter plugin stub for libclash VPN services.

This is a stub implementation that provides the API interface for the ClashMi application. It returns no-op/default responses for all methods.

## Platform Support

- Android
- iOS
- Linux
- macOS
- Windows

## Note

This is a stub implementation created to enable building ClashMi without the original proprietary libclash_vpn_service plugin.
